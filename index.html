Smart HRIS

Intelligent Human Resource Information System

WASD Weerakkodi

Bachelor of Information Technology (External Degree)

The Problem

Only 23% of Sri Lankan organizations use a comprehensive HRIS. Most rely on outdated systems and manual work.

  • Manual Data Entry

    Time-consuming, error-prone, and leads to high workloads.

  • Limited Analytics

    No predictive insights for trends like employee attrition, leading to reactive management.

  • Poor User Experience

    Outdated, non-mobile-friendly interfaces cause low adoption and frustration.

The Solution

An intelligent, AI-driven platform to automate and enhance HR processes.

  • AI-Powered Automation

    Automate resume parsing, document handling, and smart workflows.

  • Predictive Analytics

    Use machine learning to predict employee attrition and performance trends.

  • Modern UI/UX

    A clean, responsive, and mobile-first design for a great user experience.

System Architecture

A scalable, microservices approach to separate core logic from AI services.

  • Frontend

    React: For a responsive, modern web and mobile experience.

  • Backend (Core)

    Spring Boot: For robust and secure core services (payroll, leave, auth).

  • AI Services

    Python: Dedicated microservices for ML, NLP, and recommendations.
